🏥 Diabetes AI – Hybrid Predictive System

📌 Project Overview

Diabetes AI – Hybrid Predictive System is a Machine Learning-based healthcare application developed to predict the risk of diabetes using patient health information.

The project analyzes important medical parameters such as age, BMI, blood glucose level, HbA1c level, hypertension status, heart disease history, and smoking history to determine whether a patient is likely to have diabetes.

The system uses Machine Learning algorithms and a Streamlit-based web application to provide quick and user-friendly diabetes risk prediction.

---

🎯 Objectives

- Predict diabetes risk using Machine Learning techniques.
- Analyze healthcare data to identify high-risk patients.
- Improve prediction accuracy using ensemble learning.
- Provide a simple web-based interface for diabetes risk assessment.
- Support healthcare professionals with data-driven decision making.

🔄 Project Workflow

1. Data Collection

Collected the Diabetes Prediction Dataset containing patient health records.

2. Data Preprocessing

- Handled missing values
- Encoded categorical variables
- Prepared data for model training

3. Feature Engineering

Created additional risk-based features from:

- BMI
- Blood Glucose Level
- Hypertension Status

These features helped improve prediction performance.

4. Model Development

Implemented a Stacking Ensemble Model using:

- Random Forest
- XGBoost
- Logistic Regression

5. Model Evaluation

Evaluated the model using:

- Accuracy
- Precision
- Recall
- F1 Score

6. Web Application

Developed a Streamlit-based user interface for real-time diabetes risk prediction.

---

🤖 Machine Learning Models

Random Forest

Used for handling complex relationships in healthcare data and reducing overfitting.

XGBoost

Used for high-performance gradient boosting and improved prediction accuracy.

Logistic Regression

Used as a meta-classifier in the stacking ensemble model.
